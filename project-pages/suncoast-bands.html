Suncoast Bands

Description

  • Cody Banks

  • Repository

  • June 8th, 2020

A console app that stores information in a database for a record label company. Bands and Albums are stored in the database. There is a UI that allows a user to add a band, view all bands, let a band go or resign a band, view a band's albums, and a few others.

Technology

  • C#
  • LINQ
  • SQL
